# Computer parts e-commerce

## to run
```
cd e-commerce
npm install
npm run dev
```

## features
- Buy computer parts
- Look for pre-built computers
- Build your own computer

## Built with
- ReactJS
- Tailwind CSS
- Redux/rtk
- react-router
- Chakra UI
- A little bit of daisy UI
